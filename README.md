# Animal Detection & Carnivore Classification System

## Problem Statement
Detect and classify animals in images/videos and highlight carnivores.

## Features
- Multi-animal detection
- Carnivore highlighting (Red)
- GUI for image & video
- Real-time preview
- Carnivore count alert

## Model Used
YOLOv8 (pretrained on COCO dataset)

## Results
- Accurate detection of multiple animals
- Carnivore classification implemented
